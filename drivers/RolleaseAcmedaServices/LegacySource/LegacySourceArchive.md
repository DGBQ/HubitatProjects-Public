# Legacy Source Archive

This directory serves as a historical archive for the original **Rollease Acmeda Automate Pulse** drivers. These files are preserved here to provide a baseline for the logic-casting improvements and feature enhancements found in the current project root.

---

### Original Source Information
* **Original Author:** Younes Oughla
* **Last Known Update:** March 2020
* **Original Repository:** [younes-oughla/hubitat-rollease](https://github.com/younes-oughla/hubitat-rollease)

### Why is this folder separate?
These drivers have not been updated or maintained by the original author since approximately **March 2020**. While they provided the foundational structure for this integration, they are considered **deprecated** for the following reasons:

* **Logic Instability:** The original code can suffer from telnet stream interruptions and "blind reporting" sync issues on modern Hubitat firmware.
* **Limited Telemetry:** These versions do not support the advanced battery reporting or the Shade Management API logic included in the new versions.
* **Hardware Evolution:** These were designed for earlier Hubitat hub environments and may not perform optimally with current C-8 or C-8 Pro hardware capabilities.

### Usage
**These files are for reference only.** If you are looking for a stable, feature-complete integration for your Rollease Acmeda shades, please use the drivers located in the **root directory** of this repository.
