<h2 id="overview">🧭 Overview</h2>

Thank you for your interest in contributing to this repository. Whether you're improving an existing driver, suggesting enhancements, or experimenting with new ideas, contributions are always welcome.

This repository is a collection of **practical, evolving Hubitat projects**, built through hands-on experimentation and iterative learning. Contributions that improve **reliability, readability, and usability** are especially valuable.

For larger changes or new directions, you are encouraged to fork the repository and explore your own implementation — that’s very much in the spirit of this project.

---

<h2 id="project-philosophy">📜 Project Philosophy</h2>

These projects began as **AI-assisted experiments** and have grown into functional tools through testing, refinement, and real-world use.

The goal of this repository is to:
- Provide **useful and approachable solutions**
- Encourage **learning and experimentation**
- Share practical examples of **Hubitat and AI-assisted development**

Pull requests are welcome, but this repository is also intended as a **foundation for others to build upon**. If you’re inspired to take an idea further, forking and evolving your own version is strongly encouraged.
